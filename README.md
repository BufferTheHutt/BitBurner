- 🔎  [wseAutomatisierung](https://github.com/BufferTheHutt/BitBurner/blob/master/wseMoneyMaker.js): kauft Aktien sobald die Vorhersage positiv genug ist. Verkauft automatisch, wenn die Vorhersage schlecht wird.
